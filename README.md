<div align="center">

**Prerequisites:**  Node.js
 `npm install`

**Run:**
 (migrations) -> `npm run migrate`
 (front) -> `npm run dev`
 (back) -> `npm run server`

**Build:**
 `npm run build` (Builds both client and server)

**Run Production:**
 `npm start`


**NGROK Dev**
ngrok start --config ngrok.yaml --all  

**NGROK Prod**
ngrok http 3000 --config ngrok.yaml --all
